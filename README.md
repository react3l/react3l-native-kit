# react3l-native-kit
React3L Native Kit for React Native
## Installation

```sh
npm install react3l-native-kit
```

## Usage

```js
import { React3lNativeKitView } from "react3l-native-kit";

// ...

<React3lNativeKitView color="tomato" />
```
